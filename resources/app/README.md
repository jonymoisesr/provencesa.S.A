# Provencesa Invoice System

## Installation & Running Instructions

The application has been built as a portable executable. No installation is required.

1.  **Locate the Application Folder**:
    - Go to: `dist_electron/tauri-app-win32-x64`
    
2.  **Run the Application**:
    - Find the file named **`tauri-app.exe`**.
    - Double-click it to launch the invoice system.
    - You can create a shortcut to this file on your Desktop for easier access.

## Features

- **Supplier Management**: Add, edit, and delete suppliers.
- **Invoice Creation**: Create invoices with automatic calculations.
- **Invoice History**: View and manage past invoices.
- **Data Sync**: Real-time data synchronization with Firebase (Hardcoded credentials for testing).
- **Printing**: Optimized for Letter-sized printing.

## Troubleshooting

- If the app doesn't start, try running it as Administrator.
- Ensure you have an internet connection for Firebase data sync to work.
